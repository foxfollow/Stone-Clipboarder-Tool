Огляд застосунку

Ідеально вписується у ваш робочий процес macOS.

Встановлення через Homebrew

Найпростіший спосіб встановити та оновлювати.

brew tap foxfollow/stone
brew install stone-clipboarder-tool

Все що вам потрібно

Потужні функції, безкоштовно, повна приватність.

Історія буфера обміну

Відстежуйте та керуйте історією буфера обміну з постійним зберіганням за допомогою Swift Data.

Розпізнавання тексту (OCR)

Витягуйте текст із зображень за допомогою Apple Vision framework для легкого редагування та пошуку.

Розумні обрані

Закріплюйте часто використовувані елементи, захищені від автоматичного видалення. Завантаження всіх обраних.

Розширений пошук

Шукайте в історії буфера обміну за допомогою сучасної панелі пошуку з ефектом liquid glass macOS 15+.

Виключені застосунки

Вибирайте, які застосунки виключити з моніторингу буфера обміну. Ідеально для захисту конфіденційних даних.

Призупинення моніторингу

Тимчасово призупиняйте моніторинг буфера обміну з налаштовуваним таймером. Чудово для приватності.

Глобальні гарячі клавіші

Системні комбінації клавіш (⌃⌥1-0, ⌃⇧1-0) та Quick Picker для миттєвого доступу.

Режими захоплення

Вибирайте захоплення лише тексту, лише зображень або обох. Ідеально для робочих процесів Microsoft Word.

Комбіновані елементи (БЕТА)

Зберігайте текст та зображення разом як один елемент буфера обміну для керування багатим контентом.

Інтеграція в меню-бар

Швидкий доступ до елементів буфера обміну безпосередньо з меню-бар з відображенням таймера паузи.

Запуск при вході

Автоматичний запуск StoneClipboarder при вході в систему. Керуйте через налаштування доступності або Системні параметри macOS.

Налаштування доступності

Окрема вкладка налаштувань зі статусом дозволів та швидким доступом для надання доступності для автовставки та гарячих клавіш.

Автооновлення

Безпечні автоматичні оновлення за допомогою фреймворку Sparkle з підписом EdDSA.

Фокус на приватності

Усі дані залишаються на вашому пристрої. Жодної синхронізації з хмарою чи збору даних. Виключення застосунків для підвищеної приватності.

Чому обрати Stone Clipboarder?

Більше функцій. Безкоштовно. Без компромісів.

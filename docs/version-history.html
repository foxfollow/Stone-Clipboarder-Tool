Version History

Complete release notes for Stone Clipboarder Tool

Latest

Version 1.7.1

Jul 2026
  • ✨ NEW: New "Quick Picker" settings tab — Quick Look preview options plus the new type-out paste controls live here.
  • ⌨️ IMPROVED: Type-out paste speed is now adjustable (Settings › Quick Picker) — raise the per-character delay for text fields that drop characters when typing too fast.
  • ⌨️ IMPROVED: Stop an in-progress type-out anytime — press Escape or reopen the Quick Picker.

Version 1.7.0

Jul 2026
  • ✨ NEW: Type-out paste in the Quick Picker — press ⌘⇧Return to type the selected text out character by character instead of pasting, so it lands in fields that block ⌘V (secure fields, some remote desktops and terminals).
  • 🛠️ FIXED: "Show Main Window" disabled now correctly persists across cold launches — after a reboot or login-item relaunch the app no longer reappears in the Dock and Cmd+Tab when that setting is off.
  • 🧹 IMPROVED: Updated Sparkle auto-updater to 2.9.3.

Version 1.6.0

May 2026
  • 🪟 NEW: Pin clipboard items as floating windows — press ⌥P in the Quick Picker (or right-click → Pin to Screen) to keep text, images, or files on top of every Space. Pins are resizable, movable, dimmable, lockable, click-through, and collapsible, and reopen after relaunch.
  • 🔍 NEW: Zoomable image pins — pinch to zoom (centered on the cursor), drag to pan, double-click to reset; pin windows are sized to the image's aspect ratio so the image fills the window.
  • ✨ NEW: Multi-select in the Quick Picker — extend a selection with Shift+↑/↓ and paste several items at once.
  • 👁️ NEW: Multi-item OCR — press ⌥⏎ on a multi-selection to extract and combine text from images and text items.
  • ⌨️ IMPROVED: Opening the Quick Picker keeps the Settings window and your pins visible; the pin shortcuts (⌃⌥P pin most recent, ⌃⌥⇧P dismiss all) are configurable in the Hotkeys tab.
  • 🛠️ FIXED: The Quick Picker search field no longer loses focus while typing.
  • 🛡️ FIXED: Settings can no longer leave the app inaccessible — the menu bar and main window can't both be turned off.
  • 🧹 IMPROVED: Updated Sparkle to 2.9.2.

Version 1.5.5

May 2026
  • 🪟 NEW: Quick Picker tabs — filter by All / Favorites / Text / Images / Files; cycle tabs with Tab / Shift+Tab without losing search focus
  • ✨ NEW: Toggle favorite on the selected Quick Picker row with Option+Space; image rows show an OCR hint badge when OCR is enabled in Settings
  • 🛡️ NEW: Optional confirm-quit — hold ⌘Q for 1 second (or double-tap ⌘Q) to quit, with on-screen HUD; protects against accidental quits and is toggleable in Settings
  • 🎨 IMPROVED: Settings rebuilt as a sidebar NavigationSplitView; larger 720×540 minimum window size for both Settings and the main window
  • 🛠️ FIXED: SwiftData "backing data detached" crashes during auto-cleanup — favorites are now excluded from cleanup counts, the selected item is cleared before deletion, and context deletes are deferred with proper save/rollback
  • 🛠️ FIXED: Quick Look previews are now written to Application Support/StoneClipboarderTool/Previews so external apps can open them; stale preview sessions older than one hour are cleaned up on launch

Version 1.5.4

Apr 2026
  • 🛠️ FIXED: Apple Quick Look now stays selected in regular and "Fill"/maximized windows — fullscreen detection no longer falsely matches non-fullscreen full-size windows and only falls back to the Custom Preview when the focused app is in a true "Entire Screen" Space
  • ⌨️ IMPROVED: Arrow navigation while Apple Quick Look is open — the search field keeps focus and ↑/↓ now reliably moves the highlight in Quick Picker, reloading the QL preview with the new item even when QL grabs key window status

Version 1.5.3

Apr 2026
  • 🪟 NEW: Quick Picker now renders over other apps' fullscreen Spaces — open it without leaving the app you're in
  • 🖼️ NEW: Menu bar popover now appears alongside other apps' fullscreen apps too
  • 🔍 NEW: Smart preview fallback — when you trigger preview while in another app's fullscreen Space, the custom preview is used automatically (Apple Quick Look is system-managed and can't render in another app's fullscreen)
  • ⌨️ IMPROVED: Search field cursor now lands in the Quick Picker on open and is restored after closing the preview
  • 🛠️ FIXED: Quick Picker no longer pulls you out of another app's fullscreen Space when triggered
  • 🛠️ FIXED: QuickLook stability — temp files are cleaned up with a delay so "Open with…" actions have time to read them
  • 🛠️ IMPROVED: Window resignation handling — Quick Picker now closes cleanly when an external app activates (e.g. via "Open with…")
  • 🧹 IMPROVED: Code quality refactor based on SonarCloud scanner issues
